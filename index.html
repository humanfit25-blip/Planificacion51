Tras MONOLITH, llega VOLT. La roca se convierte en descarga. Semana de patrones explosivos: complexes olímpicos densos (Triple Pull lunes, Cluster + C&J pesado viernes), gimnástico avanzado (Bar Muscle-Ups, Wall Climbs, Rope Climb Skill jueves), endurance long-format diferenciado por día (chipper descendente lunes, EMOM 30 martes, Devil Intervals miércoles, intervalos 3min jueves, Hero Hansen viernes, partner pairs sábado). ◆ CrossFit
Triple Pull Complex + Barbell Ladder
▸
◆ Objetivo Apertura semana con complex olímpico de tres tirones distintos (Power Clean + Hang Power Clean + Jerk) cada 90s × 12 min. Metcon de 18 minutos en formato alternado: Set 1 barra + cardio, Set 2 técnico + run.
A) Triple Pull Complex · Every 90s × 12 min 12 MIN
  • 1 Power Clean + 1 Hang Power Clean + 1 Jerk (RPE 7 → 8, build through 8 sets)
B) Every 3 min × 18 min · Alt Set 1 / Set 2 18 MIN
  • Set 1: 9 Hang Cleans (60 / 40 kg) + 15 Burpee Box Jumps + 21 Cal Row / Bike
  • Set 2: 9 Jerks + 15 Push-Ups + 200 m Run
6 sets totales (3 × Set 1 alternando con 3 × Set 2). Score: completion of each set within 3 min.
◆ Stimulus Complex: RPE 7 → 8, NO buscar 1RM — buildar limpio. Metcon: cada set debe caber en 3 min con 20-40s descanso al final. Set 1 carga barra + cardio, Set 2 ligero técnico + run. Si Set 1 supera 2:45, escalar.
◆ RX Complex 1 PC + 1 HPC + 1 Jerk
Set 1: 9 HC (60/40) + 15 BBJ + 21 Cal
Set 2: 9 Jerks + 15 PU + 200m Run
◆ Scaled Complex 1 PC + 1 HPC + 1 PP
Set 1: 9 HC (45/30) + 12 Burpees + 15 Cal
Set 2: 9 PP + 15 Knee PU + 200m Run
◆ Feedback Coaches▸
◆ Foco Técnico
Power Clean: triple extensión, contacto cadera, codos rapidísimos. Hang Power Clean: barra en hang alta (mid-thigh), tirón explosivo, recepción en cuarto-de-squat. Jerk: dip vertical, drive cadera, lockout overhead. Cue: "barra cerca, codos altos, debajo rapidísimo".
◆ Material & Geografía
Misma barra atleta para Hang Clean y Jerks, cajas alineadas, rower/bike libre, salida del run por puerta principal. Cooridnar con halterofilia por el uso de tarimas centrales.
◆ Scaling & Banderas Rojas
Complex: si pierde recepción en Power Clean, bajar carga. Reducir peso si la zona lumbar se redondea lo más mínimo bajo fatiga.
◆ Programación
Triple Pull abre semana olímpica. Distribución correcta con el resto de la semana en empujes y tirones complejos.
